## How to View History

```bash
# View all commits with dates
git log --pretty=format:"%h - %ad - %s" --date=format:'%Y-%m-%d %H:%M'

# View graphical history
git log --oneline --graph --all

# View commits by author and date
git log --pretty=format:"%h - %an - %ad - %s" --date=short

# View commit distribution
git log --pretty=format:"%ad" --date=format:'%Y-%m' | sort | uniq -c
```
